A modern hotel management application for managing bookings, cabins, guests, and hotel operations.

## Features

- **Dashboard** - Overview of bookings, sales, check-ins, and occupancy statistics
- **Bookings Management** - View, filter, sort, and manage all reservations
- **Cabin Management** - Add, edit, delete, and duplicate cabin listings
- **Guest Check-in/Check-out** - Process arrivals and departures with extras
- **User Management** - Create internal user accounts for staff
- **Settings** - Configure breakfast prices and booking parameters
- **Dark Mode** - Toggle between light and dark themes
- **Authentication** - Protected routes with Supabase authentication

## Tech Stack

- **React 18** with React Router for navigation
- **Vite** for fast development and optimized builds
- **Styled Components** for CSS-in-JS styling
- **React Query** (@tanstack/react-query) for server state management
- **Supabase** for backend (database, authentication, storage)
- **React Hook Form** for form handling
- **Recharts** for data visualization
- **date-fns** for date manipulation
